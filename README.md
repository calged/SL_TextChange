# SCP文本修改工具

一个用于快速修改游戏本地翻译文本的工具，支持批量修改 `Class_Names.txt` 和 `Items.txt` 中的指定内容。

---

## 功能说明

- 支持修改 SCP 游戏翻译文本
- 自动创建新文件夹，避免破坏原文件
- 支持批量修改多个条目
- 不会影响原文件结构

---


## 可修改内容

1. 职业/身份标签

- SCP 实体：SCP-173、SCP-106、SCP-049、SCP-079、SCP-096、SCP-049-2、SCP-939、SCP-3114
- 基金会方人类：科学家、设施警卫、列兵、中士、指挥官、收容专家
- 混沌分裂者：D级人员、征召兵、步枪手、掠夺者、压制者
- 教程角色

2. 钥匙卡：O5级别钥匙卡、混沌分裂者破解装置
<br><br>
3. 枪械
- 手枪：COM-15、COM-18、COM-45、.44 左轮
- 冲锋枪：Crossvec、FSP-9
- 霰弹枪
- 步枪：MTF-E11-SR、AK、A7
- 轻机枪：Logicer、FR-MG-0
- 特殊：3-X 分子裂解者、囚鸟、电炮、SCP-127

4. 投掷物：高爆手雷、闪光弹、SCP-018
<br><br>
5. 医疗：急救包、肾上腺素、止痛药、SCP-500（万能药）
<br><br>
6. SCP 道具：SCP-207、SCP-268、SCP-244-A/B、SCP-1576、SCP-1853、SCP-2176、SCP-1344、SCP-1509、反向可乐
<br><br>
7. 杂项：手电筒、手提灯、糖果袋

## 使用步骤
可参考视频：https://www.bilibili.com/video/BV1YfTD6nEqj
1. 点击release下载文件;
2. 运行文本修改器.exe;
3. 打开你的 Steam，在你的库中选中 SCP 秘密实验室并单击鼠标右键，之后选择“管理” -> “浏览本地文件”，进入游戏根目录;
4. 右键 Translations 文件夹，点击复制文件地址;
5. 回到文本修改器程序，点击「选择 Translations 文件夹」按钮，粘贴刚刚复制的地址，点击选择文件夹
6. 在界面中填写需要修改的内容
7. 点击「开始修改」
8. 修改完成后在游戏中选择“游戏设置” -> “界面”-> “界面语言”，选择“简体中文1”，重启游戏后即可使用所修改的内容
9. 若需改回原始文本，可重新选择“简体中文”后重启游戏

---

## 注意事项

- 必须选择游戏根目录下的 `Translations` 文件夹
- 程序不会修改原始文本文件夹，会自动复制生成 `zh_self`文件夹进行修改
- 未填写的输入项不会被修改
